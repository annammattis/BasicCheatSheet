Border

Property Name Description
border-width: Border Width Width of the border. Same units as width and height.
border-style: Border Style Style of the border. Usual values are solid and dashed.
border-color: Border Color Color of the border. Hex, and rgb values can be used.

Background

Property Name Description
background-color: Background Color Color of the background. Takes #hex value or an rgb value.
background-image: Background Image url(URI). Takes on the path to your image.
background-repeat: Background Repeat Whether you would like the background to repeat if the width exceeds the background size.
background-position Background Position Position of the background relative to the HTML element.
