Content (Container) Tags

Tag Name Description
<h1> to <h6> Headings Headings. H1 is the main heading, H2 is secondary, etc.
<p> Paragraph Starts the instance of a Paragraph in content.
<div> Divider A container for a block of content
<span> Spanning A container for in-line content, such as content inside a paragraph.
<em> Emphasis Gives the contained text emphasis (usually as italics)
<strong> Bold Makes the contained text bold.

Document Structure

Tag Name Description
<html> HTML Tag Opens and closes an HTML document.
<head> Head Tag The first of two main sections of an HTML document. The >head> section is used to provide information about the document for use primarily by search engines and browsers.
<title> Title Tag The title of document. This element is nested inside the >head> section. In HTML5, this is the only required tag other than the DOCTYPE declaration.
<body> Body Tag The second of two main sections of an HTML document. The <body> section contains all the content of the web page.
